Introduction of Gonbike

GONBIKE is a new Bicycle sharing system that will launch in China in January of 2017 and worldwide soon after. The advances in Technology will allow a new system that will forgo the current docking system that is now in use. This will allow users the greatest flexibility to transport themselves from one place to another without requiring to identify a docking system to return the bike too.
Now User will be able to leave the bike anywhere in the city for their convenience.
GONBIKE was created to improve public transportation by making it more healthy, flexible for the user and good for the environment.
