【客服经理】
岗位职责

1. 根据公司战略发展方向，制定各城市及区域的整体客服体系；
2. 负责客服团队的构建和日常管理，提升团队业务能力；
3.制定有效的客服工作激励机制，不断提高团队的服务水平；
4. 参与客服、审核、指令派发等各模块的工作，熟知各部门业务流程；
5. 妥善处理客户反馈的事项，追求卓越的客户满意度。

任职要求：

1. 全日制本科及以上学历；
2. 三年以上专业客服及客服经理工作经验；
3. 有大型客服团队管理经验者优先考虑。
【客服专员】
岗位职责

1. 协助运营主管建设客服团队、培训坐席，以对用户负责的态度开展客服工作；
2. 协助策划、建立、完善客服管理部实施方案、管理制度、客服标准及流程；
3. 协助完成客服管理部门服务流程的成熟化运作、客户满意度的持续提高；
4. 完成上级安排的其他工作任务。

任职要求：

1. 中专及以上学历，应届毕业亦可；
2. 普通话标准；
3. 会基本的电脑操作，打字每分钟40字以上；
4. 有客服经验者优先考虑。
【车辆运维经理】
岗位职责

1. 负责城市车辆运维团队的构建和日常管理，不断提高车辆运维能力；
2. 负责城市车辆的统一投放、故障处理和区域协调，及时救援特定车辆；
3. 分析城市车辆运营数据，优化区域车辆投放策略，不断提高车辆使用率；
4. 收集同城竞争对手信息，实时掌握竞争对手动态，并提供竞争参考建议。

任职要求：

1. 全日制本科及以上学历；
2. 能吃苦耐劳，愿意全身心投入工作，确保城市车辆体系正常运行；
3. 有线下推广、团队管理经验者优先考虑。
【车辆运营专员】
岗位职责

1. 负责城市车辆的投放、故障处理和救援；
2. 负责区域点位的车辆日常维护。

任职要求：

1. 大专及以上学历，应届毕业亦可；
2. 能吃苦耐劳，愿意全身心投入工作，确保城市车辆体系正常运行；
3. 有线下推广、市场拓展经验者优先考虑。
【市场专员】
岗位职责

1. 负责公司产品的市场推广工作，肩负用户获取和提升产品知名度；
2. 负责公司项目微信、微博、官网等自媒体的管理，完成传播策划和发布任务；
3. 负责BD拓展和洽谈，策划市场合作方案，不断优化品牌传播。

任职要求：

1. 全日制本科及以上学历；
2. 2年及以上市场营销、活动策划、BD拓展等相关工作经验；
3. 擅长数据分析，有完善的分析报告尤佳；
4. 富于创造性, 喜欢用创新的方法来设计有趣的市场活动来获取用户；
5. 有自己运营的社交媒体账号者优先考虑。
【媒体公关经理】
岗位职责

1. 负责公司媒体和政府公共关系，制定媒体/公关计划、策略及执行；
2. 制定公关活动效果评估标准，并参与考核；
3. 独立撰写各类市场活动所需宣传稿件、博文、论坛帖；
4. 与相关媒体保持良好关系，进行媒体资源整合和维护；
5. 负责与政府相关部门关系的建立，拓展，游说；
6. 统筹并游说政府政策协调，为公司项目运营提供政府支持；
7. 维护和提升公司品牌，处理公关危机；
8. 竞争对手及同行公关运作手法的跟踪和分析。

任职要求：

1. 全日制本科以上学历，公关、新闻等相关专业优先；
2. 三年以上媒体公关工作经验，大型公关公司或一线互联网公司同等职位经验者优先；
3. 拥有丰富的媒体资源，能够熟练撰写新闻稿、发言稿以及其他公关所需文件；
4. 工作主动性强，具有业务专研精神。
